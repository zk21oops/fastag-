A simple, secure, and modern FASTag recharge platform designed with the **Serene Confidence** design system.  
This project allows users to log in, register vehicles, recharge FASTag wallets, and review their transaction history.  
It also supports app conversion via **PWA** or **WebView**.
  Features
- User Authentication (Login/Signup)
- Vehicle FASTag Details Management
- Recharge Flow With Payment Gateway
- Transaction/Recharge History
- Notifications
- Conversion to Mobile App (PWA / WebView support)
- Clean & accessible UI
